Logbook

Logbooks are a record of important events and are often used for projects such as this Portfolio Website. In my logbook, I am noting down any changes which are done to either the portfolio website or documentation with four columns of information. The first column is Date, when was the entry added into the logbook? The second column is Time, how long did this take to complete or do? The third column is Where, was this work done in class or at home? The fourth column is Action, what was exactly done? This part has a bunch of text, images, links, etc.
